### Where can I ask for help or get more information?

For questions or more information, please contact:

* [help@openva.net](mailto:help@openva.net) 

* [info@openva.net](mailto:info@openva.net)


Please submit openVA issues for bugs or requests on GitHub at 

* [GitHub issues for openVA (R version)](https://github.com/verbal-autopsy-software/openVA/issues)

* [GitHub issues for (Python) openVA App](https://github.com/verbal-autopsy-software/pyopenva_GUI/issues)


### Where can I get more information about openVA?

The publication on [The openVA Toolkit for Verbal Autopsies](https://journal.r-project.org/articles/RJ-2023-020/openVA-RJ-R1.pdf) by Richard Li and the openVA Team
is a useful resource for the R implementation of openVA, which includes

* an overview of several VA algorithms (including InSilicoVA and InterVA)

* a description of the data formats

* examples of using the openVA to assign causes of death and examine the results
