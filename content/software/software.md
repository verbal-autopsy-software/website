### OpenVA Pipeline
+ Software for automating the processing of verbal autopsy data : [PyPI](https://pypi.org/project/openva-pipeline/), [GitHub](https://github.com/verbal-autopsy-software/openva_pipeline)
+ [Documentation](https://openva-pipeline.readthedocs.io/en/latest/)
+ Installation Tutorial: [Youtube](https://www.youtube.com/watch?v=C2EPOpTzJTk)
+ Analysis Guide: [Youtube](https://www.youtube.com/watch?v=K1wkSbTwxkg)


### openVA
+ openVA is the main package for VA analysis in R: [CRAN](https://cran.r-project.org/web/packages/openVA/index.html), [GitHub](https://github.com/verbal-autopsy-software/openVA)
+ Graphic User Interface (GUI) application in Python: [GitHub](https://github.com/verbal-autopsy-software/pyopenva_GUI)

### CrossVA
+ crossVA, for data preparation from ODK in R: [CRAN](https://cran.r-project.org/web/packages/CrossVA/index.html), [GitHub](https://github.com/verbal-autopsy-software/CrossVA)
+ pyCrossVA, for data preparation from ODK in Python: [PyPI](https://pypi.org/project/pycrossva/), [GitHub](https://github.com/verbal-autopsy-software/pyCrossVA)


### VA Algorithms
+ InSilicoVA: [CRAN](https://cran.r-project.org/web/packages/InSilicoVA/index.html), [GitHub](https://github.com/verbal-autopsy-software/InSilicoVA)
+ InterVA4: [CRAN](https://cran.r-project.org/web/packages/InterVA4/index.html), [GitHub](https://github.com/verbal-autopsy-software/InterVA4)
+ InterVA5: [CRAN](https://cran.r-project.org/web/packages/InterVA5/index.html), [GitHub](https://github.com/verbal-autopsy-software/InterVA5)
+ Tariff: [CRAN](https://cran.r-project.org/web/packages/Tariff/index.html), [GitHub](https://github.com/verbal-autopsy-software/Tariff)
+ NBC: [CRAN](https://cran.r-project.org/web/packages/nbc4va/index.html), [GitHub](https://github.com/rrwen/nbc4va)
